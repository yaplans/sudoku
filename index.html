Справка

1. Выберите интересующий Вас семестр (по умолчанию выбран последний).

Период будет изменен. Если нужно, можно откорректировать период, с которым программа будет работать.

2. Выберите интересующие Вас дисциплину и группу.

В скобках указаны количество дисциплин и групп попавших в выбранный период. После выбора дисциплины в списке групп перед названием каждой группы появляется знак '+' или '-'. Знак '+' означает, что у этой группы и дисциплины были занятия. Точно также, при выборе группы, в списке дисциплин будет указано у какой дисциплины были занятия. Если у выбранных группы и дисциплины будет знак '+', данные в таблице будут обновлены.

3. По умолчанию, данные в журнале находятся в режиме "чтение".

Чтобы перейти в режим "редактирование", нужно изменить положение соответствующего переключателя.

4. Для просмотра тем занятий нужно нажать на кнопку "Показать темы".

Темы выбираются при создании занятия, но можно изменить выбранную тему нажав на кнопку с символом "карандаш".

Примечание. Темы для дисциплин настраиваются отдельно, преподаватель может только выбрать необходимую тему из предложенного списка.

5. Кнопка "Добавить занятие" вызовет форму, заполнив которую, можно сохдать занятие для выбранной группы и дисциплины.

6. Кнопка "Удалить занятие" отобразит рядом с датами занятий значки с символом "корзина", предназначенные для удаления занятий.

7. Для завершения занятия необходимо нажать на соответствующуб кнопку.
